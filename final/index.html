Addendum: Lights Out Game

This project was created to practice core web development skills. It involved building an interactive "Lights Out" puzzle game using HTML, CSS and JavaScript.

HTML:

  • Structured the page with semantic elements like <header>, <div>, <button>, and <footer>.
  • Used <audio> tags for sound effects and <script> tags to include logic.

CSS:

  • Designed a responsive grid layout for the lights using Flexbox.
  • Styled the buttons, header, and win message with animations (e.g., blinking heading).
  • Created hover effects, color changes, and visual feedback for game states.

JavaScript:

  • Generated a 5x5 game grid dynamically using the DOM.
  • Handled click events to toggle lights and check for win conditions.
  • Implemented two difficulty levels: Easy (individual toggle) and Hard (plus neighbors).
  • Used audio feedback on interactions and controlled visibility of elements like the Addendum.

Conclusion: This project helped strengthen my understanding of dynamic DOM manipulation, interactive design with CSS and logical control flow in JavaScript.
